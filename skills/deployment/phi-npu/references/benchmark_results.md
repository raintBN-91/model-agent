## 测试 prompt

```
1. "def fibonacci(n):"
2. "Machine learning is a field of"
3. "Write a Python function to sort a list:"
4. "The capital of France is Paris."
5. "Quantum computing is"
```

## 结论

1. 三个模型均达到 **Top-1 100%** 一致，余弦相似度 > 0.999，KL 散度远低于 0.01 阈值
2. Phi-1 的 fp32→fp16 转换引入的数值误差极小（KL ~3e-8），不影响推理质量
3. 所有模型可稳定运行在 1x Ascend910B4 上，显存充足
