# 验证硬门禁

## H1 功能门禁（必须全通过）

- [ ] 模型注册/构建成功
- [ ] 数据注册/构建成功
- [ ] 配置三元关联一致
- [ ] 一次分布式 E2E 成功

## H2 可靠性门禁（必须全通过）

- [ ] 路径类型兼容（str/list[str]）
- [ ] strict/extra 分层检查通过
- [ ] 加载签名兼容检查通过
- [ ] 错误配置报错可读且可执行修复

## H3 证据门禁（必须全通过）

- [ ] 每项门禁都有命令与退出码
- [ ] evidence.json 字段完整且可追溯
- [ ] 失败项包含回流技能与修复动作

## 失败规则

- [ ] 任一硬门禁失败即整体失败
- [ ] 不允许“部分通过后宣称上线可用”
