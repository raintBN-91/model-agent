# AI4S 模型类型提示

## 目的

不同 AI4S 模型的“应该比较什么”并不一样。进入对齐前，先判断模型类型，避免拿错指标或输出形式。

## 常见类型与优先比较项

### PDE 或 CFD 类模型

优先比较：

- 训练 `loss`
- 验证集误差
- 关键物理量场的逐样本误差
- 长时滚动预测是否逐步发散

重点关注：

- 网格或坐标预处理
- 边界条件编码
- 时间步推进中的累积误差

### 分子性质预测或材料性质预测

优先比较：

- MAE、RMSE、R2
- 逐样本预测值偏差
- 图构建或特征提取结果

重点关注：

- 原子或键特征构造
- 标准化与反标准化
- batch 内 padding 或 mask

### 图神经网络类模型

优先比较：

- 图级或节点级指标
- logits 或中间 embedding
- 稀疏邻接结构相关输入

重点关注：

- 图构图顺序
- 邻接矩阵或边索引排序
- scatter 或 gather 相关算子行为

### 时序预测类模型

优先比较：

- MAE、MSE、MAPE
- 单步预测和多步滚动预测
- 长序列后半段是否开始漂移

重点关注：

- 滑窗切分
- 时间特征编码
- teacher forcing 与解码步逻辑

### 生成式模型

优先比较：

- 确定性设置下的 token 级输出
- logits 或概率分布
- 最终文本

重点关注：

- tokenizer
- prompt 模板
- sampling 参数
- KV cache 或 decode step 行为

## 使用原则

- 先按模型类型选比较对象，再运行脚本。
- 如果用户没有说明模型类型，先问清任务类型和最终指标是什么。
- 如果模型是混合任务，不要只看一个指标，要同时保留任务指标和中间输出证据。
