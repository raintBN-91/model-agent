# 电力时序预测智能体

## 项目概述
基于昇腾NPU构建的电网时序预测智能体，提供数据治理、模型训练、推理预测、效果评估及持续优化的全流程能力。支持台区负荷与新能源出力预测，实现轻量化、高精度、可复用的预测工作流，助力电力调度与用能优化。

## 核心功能
- **全流程预测能力**：提供电力时序预测全流程能力，包括数据清洗、模型训练、推理部署与效果评估
- **国产化算力支持**：适配昇腾NPU轻量化推理，支持国产化算力
- **效果优化**：降低预测偏差，提升电网调度效率与节电效果

## 快速使用

### 1. 环境依赖
```bash
# 昇腾 CANN
# PyTorch-NPU  
# 时序数据处理依赖库
```

### 2. 技能调用流程
依次加载以下技能模块：
- `data_prepare` → `train` → `infer` → `evaluate` → `optimize`

### 3. 全流程运行
执行 `forecast_workflow`，一键完成从数据到预测优化全链路

### 4. 效果查看
输出内容包括：
- 预测结果
- 误差指标  
- 节电收益评估报告

## 项目信息
