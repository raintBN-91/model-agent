# 测试用例生成规则

## 1. 用例规模要求

| 精度等级 | 用例规模 | 每种dtype数量 | 输出总元素数 |
|---------|---------|-------------|------------|
| **L0** | ≥5,000 | ≥200 | ≥1,000,000 |
| **L1** | ≥10,000 | ≥700 | ≥1,000,000 |
| **L2** | ≥30,000 | ≥700 | ≥1,000,000 |

## 2. Tensor生成规则

### 2.1 数据类型

- 覆盖所有支持数据类型
- L0: 每种类型≥200
- L1/L2: 每种类型≥700

### 2.2 数据格式

覆盖所有支持格式：ND/NCHW/FRACTAL_NZ等

### 2.3 数据维度

- 维度数: 1-8
- 维度值: 1-2^31
- 总元素数 ≤ 2^31

### 2.4 数据值域分布

**L0级：**
| 分布类型 | 参数 | 占比 |
|---------|-----|------|
| 均匀分布 | [-5, 5] | 50% |
| 正态分布 | μ∈[-100,100], σ∈[1,25] | 50% |

**L1级：**
| 分布类型 | 参数 | 占比 |
|---------|-----|------|
| 均匀分布 | [-0.001, 0.001] | 10% |
| 均匀分布 | [-5, 5] | 30% |
| 正态分布 | μ∈[-100,100], σ∈[1,25] | 40% |
| 离群点分布 | 0.1%值×1000 | 20% |

**L2级：**
在L1基础上新增模型真实输入（泛化:模型=2:1）

## 3. Attr生成规则

| 参数类型 | 生成规则 |
|---------|---------|
| 标量 | 覆盖所有等价类场景 |
| 布尔 | 覆盖True和False |
| 枚举 | 覆盖所有支持枚举值 |
| 组合 | 参数类型进行组合遍历 |

## 4. 特殊场景测试

必须覆盖但不计入规模：

| 场景 | 定义 | 生成规则 |
|-----|------|---------|
| 空Tensor | 某维度为0 | 每种dtype每个dim遍历 |
| 标量Tensor | shape=[1] | 每种dtype一个 |
| 上下边界 | dim=1或2^31+1 | 边界值测试 |
| INF/NAN | 特殊值 | 每种dtype每种值4个shape |
| 异常值 | 边界外/约束外 | 防护和拦截验证 |

## 5. 离群点分布生成

### 5.1 规则

```
k = max(1, floor(n / 1000))  # 离群点数量
随机选取k个索引
这些位置的值 × 1000
```

### 5.2 示例

```python
X = [[0.1, 0.2, 0.3], [-0.1, -0.2, -0.3]]
k = 1  # 6个元素，floor(6/1000)=1
X_noisy = [[0.1, 0.2, 0.3], [-100.0, -0.2, -0.3]]
```

## 6. 正反向级联测试

**适用：** 训练中的反向算子

**方法：** 设计正向+反向组合测试用例

## 7. 生成注意事项

1. 数据类型、格式、维度需正交
2. 输出总元素≥1,000,000
3. 特殊场景不计入规模
4. 不同随机种子视为相同用例
